LV1-1：安装java环境&第一个java程序

一、java环境组成

JRE：java运行时环境，包含了java虚拟机，java基础类库。是使用java语言编写的程序运行所需要的软件环境，简单来说，如果你只是想运行你的java程序，只安装JRE就行了。

JDK：java开发工具包，提供了全套JRE、编译器、一些开发工具包，如果你想要开发java程序，需要安装一个JDK。
